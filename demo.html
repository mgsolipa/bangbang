Bang Bang

jQuery plugin to handle multiple tagged input.

What is this?

Bang Bang is a way to group inputs on a single input field. It's sort of like how tags work except differently. It was originally written for our advanced search functionality. The goal was a simple way to provide our support team with a fast way to search for information and not have them click a bunch of buttons, and checkboxes to filter their search criteria.

Why is it called Bang Bang?

Because of the exclamation point ! is what triggers the commands.

Does it work with bootstrap?

Yes, it should work with version 2.2.1

How does it work?

When you type ! followed by the command, a new text input field is added to the form with the name of the command. It doesn't look like it from the UI but it happens behind the scenes. You can call the jquery plugin with a list of tags like this:

    <script type="text/javascript">
      $(document).ready(function() {
        $(".help_link").click(function(e) {

        var tags = [
          "idnumber",
          "f_name",
          "m_name",
          "l_name",
          "gender",
          "test1",
          "test2"

        var tagsTitles = [
          "ID Number",
          "First name",
          "Middle name",
          "Last name",
          "Gender",
          "Long tag test 1",
          "Long tag test 2"
        ];
        
        $("#search_txt").bang({ tags: tags, tagsTitles: tagsTitles });

        $("#search_button").click(function(e) {
          e.preventDefault();
          $("#form").submit();
        });
      });
